# BE
NodeJS Express App for School in the Cloud


## User Requirements:

- a unique `email`
- a `password`
- a `name`

Here is a table with the `endpoint` descriptions:

| Action                     | URL                     | Method | Response                  |
| :------------------------- | :---------------------- | :----- | :------------------------ |
| Register a user            | /api/auth/register      | POST   | the new user + a token    |
